> NIM_UPDATEIOS    (/home/travis/.ansible/plugins/modules/nim_updateios.py)

        Performs updates and customization to the Virtual I/O Server
        (VIOS).

  * This module is maintained by The Ansible Community
OPTIONS (= is mandatory):

- accept_licenses
        Specifies whether the software licenses should be
        automatically accepted during the installation.
        [Default: (null)]
        type: str

= action
        Operation to perform on the targets.
        `install'.
        `commit'.
        `reject'.
        `cleanup'.
        `remove'.
        (Choices: install, commit, reject, cleanup, remove)
        type: str

- filesets
        Specifies a list of file sets to remove from the target.
        [Default: (null)]
        type: str

- installp_bundle
        Specifies an `installp_bundle' resource that lists file sets
        to remove on the target.
        [Default: (null)]
        type: str

- lpp_source
        Identifies the `lpp_source' resource that will provide the
        installation images for the operation.
        [Default: (null)]
        type: str

- nim_node
        Allows to pass along NIM node info from a task to another so
        that it discovers NIM info only one time for all tasks.
        [Default: (null)]
        suboptions: null
        
        type: dict

- preview
        Specifies a preview operation.
        [Default: (null)]
        type: str

= targets
        NIM targets.

        type: str

- time_limit
        Before starting the action, the actual date is compared to
        this parameter value; if it is greater then the task is
        stopped; the format is `mm/dd/yyyy hh:mm'.
        [Default: (null)]
        type: str

- vars
        Specifies additional parameters.
        [Default: (null)]
        suboptions:
          log_file:
            description:
            - Specifies path to log file.
            type: str
        
        type: dict

- vios_status
        Specifies the result of a previous operation.
        [Default: (null)]
        suboptions: null
        
        type: dict


REQUIREMENTS:  AIX >= 7.1 TL3, Python >= 2.7

AUTHOR: AIX Development Team (@pbfinley1911)
        METADATA:
          status:
          - preview
          supported_by: community
        

EXAMPLES:

- name: Update a pair of VIOSes
  nim_updateios:
    targets: "(nimvios01, nimvios02)"
    action: install
    lpp_source: /lpp_source
