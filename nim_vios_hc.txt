> NIM_VIOS_HC    (/home/travis/.ansible/plugins/modules/nim_vios_hc.py)

        Check if a pair of Virtual I/O Servers can be updated.

  * This module is maintained by The Ansible Community
OPTIONS (= is mandatory):

= action
        Specifies the operation to perform.
        `health_check' to perform a health check.
        (Choices: health_check)
        type: str

= targets
        NIM target.
        To perform a health check on dual VIOSes specify a tuple with
        the following format: "vios1,vios2".

        elements: str
        type: list

- vars
        Specifies additional parameters.
        [Default: (null)]
        type: dict


NOTES:
      * Requires vioshc.py available at
        https://github.com/aixoss/vios-health-checker.


REQUIREMENTS:  AIX >= 7.1 TL3, Python >= 2.7

AUTHOR: AIX Development Team (@pbfinley1911)
        METADATA:
          status:
          - preview
          supported_by: community
        

EXAMPLES:

- name: Perform a health check on dual VIOSes vios1,vios2 and on VIOS vios3
  nim_vios_hc:
    targets:
    - vios1,vios2
    - vios3
    action: health_check


RETURN VALUES:

msg:
    description: Status information.
    returned: always
    type: str
targets:
    description: List of VIOS tuples.
    returned: always
    type: list
    elements: str
nim_node:
    description: NIM node info.
    returned: always
    type: dict
status:
    description: Status for each VIOS (dictionary key).
    returned: always
    type: dict
